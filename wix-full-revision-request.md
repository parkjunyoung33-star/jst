# JST 홈페이지 전체 리뉴얼 요청서

## 리뉴얼 목표

현재 Wix AI가 만든 홈페이지는 기본 구조는 괜찮지만, 전체적으로 회색 배경과 딱딱한 문구 때문에 프리미엄 스포츠 에이전시의 신뢰감과 글로벌 이미지를 충분히 주지 못합니다.

JST는 단순한 스포츠 에이전시가 아니라, 배구를 시작으로 다양한 종목의 선수 커리어를 글로벌 무대로 확장하는 스포츠 매니지먼트 회사처럼 보여야 합니다.

## 브랜드 톤

- 신뢰감
- 글로벌
- 전문적
- 프리미엄
- 선수 중심
- 과하게 화려하지 않고 세련된 스포츠 비즈니스 느낌

피해야 할 느낌:

- 동호회 느낌
- 일반 회사 소개 템플릿 느낌
- 너무 회색이고 딱딱한 제안서 느낌
- 스톡 이미지가 과한 느낌
- 실제 소속 선수가 아닌 이미지로 소속 선수처럼 보이는 구성

## 1. Hero 섹션

현재 회색 배경은 너무 밋밋합니다.

수정 방향:

- 배경은 어두운 스포츠 경기장 이미지, 선수 실루엣, 코트, 에이전시 미팅 느낌의 고급 이미지 중 하나를 사용
- 텍스트는 왼쪽 정렬
- 제목은 2줄 정도로 강하게
- 긴 문단은 줄이고 버튼 2개만 배치

## 3. Services 섹션

현재 서비스 카드는 검은 테두리와 회색 박스 때문에 조금 오래된 느낌이 납니다.

수정 방향:

- 카드 테두리는 없애거나 아주 연하게
- 배경은 흰색 또는 아주 밝은 회색
- 아이콘은 작고 정돈되게
- 카드 높이와 간격을 통일
- 문구는 짧고 명확하게

제목:

```text
선수의 커리어를 설계하는 JST의 전문 서비스
```

서비스 카드:

```text
계약 및 이적 전략
국내외 리그와 팀 니즈를 분석해 선수에게 가장 적합한 계약 방향을 설계합니다.

글로벌 커리어 로드맵
현재의 경기력과 장기 목표를 바탕으로 다음 무대를 준비합니다.

브랜드 파트너십
선수의 이미지와 팬덤을 기반으로 브랜드 협업 기회를 만듭니다.

미디어 매니지먼트
인터뷰, 콘텐츠, 소셜 채널을 통해 선수의 메시지를 일관되게 관리합니다.
```

## 4. Values 섹션

현재 소통, 신뢰, 성과 섹션은 방향은 좋지만 레이아웃이 조금 무겁고 텍스트 배치가 어색합니다.

수정 방향:

- 검은 배경을 유지해도 좋지만 여백을 더 정리
- 세 개의 가치를 같은 높이와 같은 구조로 배치
- 설명 문구는 너무 길지 않게
- 세로선 장식은 줄이거나 제거

제목:

```text
JST가 지키는 세 가지 기준
```

내용:

```text
소통
선수와의 깊은 이해를 바탕으로 커리어의 방향을 함께 결정합니다.

신뢰
모든 계약과 의사결정에서 정직하고 투명한 파트너가 됩니다.

성과
경기장 안팎에서 선수의 가치가 실제 결과로 이어지도록 지원합니다.
```

## 5. Talent Portfolio 섹션

현재 "소속 선수" 섹션에 실제 소속 선수가 아닌 이미지가 들어가 있다면 신뢰도에 문제가 생길 수 있습니다.

아직 선수 자료가 준비되지 않았다면 "소속 선수"라는 표현을 쓰지 말고 아래처럼 구성해주세요.

제목:

```text
TALENT PORTFOLIO
```

큰 문구:

```text
선수 한 명 한 명이 하나의 브랜드로 보이도록
```

본문:

```text
JST는 선수의 기록, 스토리, 포지션, 성과, 미디어 활동을 체계적으로 정리해 커리어 가치를 보여주는 포트폴리오를 구축합니다.
```

카드 3개:

```text
Volleyball
JST의 시작점인 배구 선수 매니지먼트를 중심으로 전문성을 쌓아갑니다.

Multi-Sport
향후 다양한 종목의 선수들이 같은 기준으로 소개될 수 있는 구조를 준비합니다.

Global Pathway
국내를 넘어 해외 무대와 브랜드 기회까지 연결하는 방향을 지향합니다.
```

실제 선수 데이터가 들어온 후에는 이 섹션을 선수 카드 목록으로 변경합니다.

## 6. Media 섹션

현재 미디어 이미지는 AI/스톡 이미지 느낌이 강합니다.

수정 방향:

- 실제 기사, 인터뷰, 경기 영상, SNS 콘텐츠가 생기기 전까지는 너무 많은 가짜 콘텐츠를 넣지 않기
- 카드 개수를 3개 이하로 유지
- "최신 뉴스"보다 "JST Media" 또는 "Media Presence"처럼 에이전시 브랜드 느낌으로 구성

제목:

```text
MEDIA PRESENCE
```

큰 문구:

```text
뉴스, 인터뷰, 하이라이트가 선수의 가치로 이어지도록
```

본문:

```text
JST는 선수의 경기력뿐 아니라 미디어 노출과 콘텐츠 흐름까지 관리하며, 커리어의 중요한 순간이 더 많은 사람들에게 전달될 수 있도록 돕습니다.
```

## 디자인 수정 요청

- 전체적으로 회색 면적을 줄이고, 흰색/검정/딥네이비/블루 포인트 중심으로 정리
- 파란색은 너무 쨍하지 않게 사용
- 카드 테두리는 검정색 두꺼운 선 대신 얇고 연한 선 사용
- 폰트는 굵게만 쓰지 말고 제목/본문의 대비를 자연스럽게
- 본문 문단 폭은 너무 넓지 않게
- 첫 화면에는 반드시 강한 비주얼 이미지 사용
- 실제 소속 선수가 아닌 이미지는 "소속 선수"처럼 보이지 않게 처리
- 모바일에서 제목이 너무 크게 깨지지 않도록 확인
