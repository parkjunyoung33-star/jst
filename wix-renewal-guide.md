# JST Wix Renewal Guide

## Brand Direction

JST should be positioned as a trust-led global sports management agency.

Core message:
`신뢰를 기반으로 선수의 무대를 세계로 확장합니다.`

Supporting message:
`JST는 배구를 시작으로 다양한 종목의 선수 커리어, 계약, 브랜딩, 미디어 가치를 함께 설계하는 글로벌 스포츠 매니지먼트 파트너입니다.`

## Recommended Site Structure

1. HOME
   - Hero
   - About summary
   - Services
   - Talent preview
   - Media preview
   - Contact

2. ABOUT
   - JST mission
   - Communication / Trust / Performance
   - Global expansion message

3. SERVICES
   - Contract and transfer strategy
   - Global career roadmap
   - Brand partnership
   - Media management

4. TALENT
   - Athlete grid
   - Sport filter
   - Each athlete links to a player detail page

5. PLAYER DETAIL
   - Name
   - Sport
   - Position
   - Team
   - Achievements
   - Profile image
   - Match image
   - Video / article / SNS links
   - One-line brand story

6. MEDIA
   - News
   - Interviews
   - Highlights
   - Social content

7. CONTACT
   - Athlete inquiry
   - Club inquiry
   - Brand partnership inquiry

## First Build Order In Wix

1. Duplicate or preserve the existing site before heavy edits.
2. Create the new HOME page first.
3. Rebuild ABOUT using the existing website content.
4. Create SERVICES as a new page or HOME section.
5. Create the Athletes CMS collection.
6. Build TALENT list page from the CMS.
7. Build PLAYER DETAIL dynamic page.
8. Add MEDIA CMS later.
9. Review mobile layout.
10. Publish only after final confirmation.

## Current Content Sufficiency

It is not enough for a complete player portfolio yet.

For the first version, use:
