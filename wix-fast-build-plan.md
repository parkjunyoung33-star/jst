# JST Wix 빠른 제작 플랜

## 결론

현재 Wix AI가 만든 페이지를 하나씩 수정하는 방식은 비효율적입니다.

가장 빠른 방법은 기존 AI 초안을 계속 고치는 것이 아니라, 새 홈 페이지를 거의 빈 화면에서 다시 만드는 것입니다.

## 추천 방식

1. 기존 AI 페이지는 삭제하지 말고 그대로 둔다.
2. Wix에서 새 페이지를 만든다.
3. 페이지 이름을 `Home Renewed` 또는 `New Home`으로 둔다.
4. 섹션은 딱 6개만 만든다.
5. 각 섹션에 아래 문구를 그대로 붙인다.
6. 완성 후 기존 홈을 새 홈으로 교체한다.

## 필요한 섹션

### 1. Header

로고:
`JST`

메뉴:

```text
About
Services
Talent
Media
Contact
```

버튼:

```text
문의하기
```

삭제:

```text
Inquiry Services Page
서비스 문의
```

`Inquiry Services Page`는 메뉴에서 빼는 것이 좋습니다.

## 2. Hero

작은 문구:

```text
GLOBAL SPORTS MANAGEMENT AGENCY
```

제목:

```text
신뢰를 기반으로
선수의 무대를 세계로 확장합니다.
```

본문:

```text
JST는 배구를 시작으로 다양한 종목의 선수 커리어, 계약, 브랜딩, 미디어 가치를 함께 설계하는 스포츠 매니지먼트 파트너입니다.
```

버튼:

```text
선수 문의
파트너십 문의
```

디자인:

- 흰 배경만 쓰면 너무 비어 보입니다.
- 어두운 스포츠 경기장 이미지 또는 딥네이비 배경을 사용합니다.
- 인물 단독 사진은 쓰지 않습니다.
- 제목은 너무 크게 하지 않습니다.

## 3. About

제목:

```text
좋은 에이전시는 계약을 돕고,
상위 에이전시는 선수의 시장을 넓힙니다.
```

본문:

```text
JST는 선수의 가능성을 믿고, 커리어의 중요한 순간마다 함께 판단하고 움직이는 스포츠 매니지먼트 파트너입니다. 배구를 시작으로 국내외 스포츠 시장을 연결하며, 선수들이 더 넓은 무대에서 자신의 가치를 증명할 수 있도록 지원합니다.
```

포인트:

```text
Career Strategy
선수의 현재와 미래를 함께 고려한 커리어 방향을 설계합니다.

Global Network
국내외 스포츠 시장의 흐름을 이해하고 더 넓은 기회를 연결합니다.

Brand Value
경기력뿐 아니라 미디어, 이미지, 팬덤까지 선수의 가치를 확장합니다.
```

## 4. Services

제목:

```text
선수의 커리어를 설계하는 JST의 전문 서비스
```

카드 4개:

```text
계약 및 이적 전략
국내외 리그와 팀 니즈를 분석해 선수에게 가장 적합한 계약 방향을 설계합니다.

글로벌 커리어 로드맵
현재의 경기력과 장기 목표를 바탕으로 다음 무대를 준비합니다.

브랜드 파트너십
선수의 이미지와 팬덤을 기반으로 브랜드 협업 기회를 만듭니다.

미디어 매니지먼트
인터뷰, 콘텐츠, 소셜 채널을 통해 선수의 메시지를 일관되게 관리합니다.
```

디자인:

- 사진 카드 사용하지 않기
- 회색 박스 줄이기
- 카드 배경 흰색
- 테두리는 아주 연하게
- 아이콘은 없어도 됩니다

## 5. Values

제목:

```text
JST가 지키는 세 가지 기준
```

내용:

```text
소통
```

- 세 카드만 배치
- 텍스트는 짧게

## 6. Talent Portfolio

제목:

```text
TALENT PORTFOLIO
```

큰 문구:

```text
선수 한 명 한 명이 하나의 브랜드로 보이도록
```

본문:


카드:

```text
Volleyball
JST의 시작점인 배구 선수 매니지먼트를 중심으로 전문성을 쌓아갑니다.

Multi-Sport
향후 다양한 종목의 선수들이 같은 기준으로 소개될 수 있는 구조를 준비합니다.

Global Pathway
국내를 넘어 해외 무대와 브랜드 기회까지 연결하는 방향을 지향합니다.
```

주의:

실제 선수가 아닌 이미지를 `소속 선수`처럼 보여주지 않습니다.

## 작업 시간 줄이는 방법

### 하지 말 것

- Wix AI가 만든 현재 섹션을 계속 고치기
- 가짜 선수 사진 고르기
- 서비스 카드마다 사진 넣기
- 문구를 계속 새로 생성하기
- 메뉴를 너무 많이 만들기

### 할 것

- 새 페이지에서 6개 섹션만 만들기
- 문구는 위 내용을 그대로 사용하기
- 이미지는 Hero 한 곳만 신경 쓰기
- 선수 자료는 나중에 CMS로 추가하기
- 게시 전 모바일만 확인하기

## 가장 빠른 Wix 작업 순서

1. 새 페이지 만들기
2. 빈 섹션 6개 추가
3. Header 정리
4. Hero 문구 붙이기
5. Services 카드 4개 만들기
6. Values 카드 3개 만들기
7. Talent Portfolio 카드 3개 만들기
8. Contact 섹션 만들기
9. 모바일 확인
10. 새 페이지를 홈으로 지정

## 판단

지금 AI 초안은 버리고, 이 플랜으로 새로 만드는 것이 더 빠릅니다.
기존 페이지를 수정하면 시간이 계속 늘어납니다.
